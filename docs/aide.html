Guide d'utilisation de l'application

Qu'est-ce que l'application ?

1 / 8

Objectif

  • Suivre les kits de pièces pour l'aciérie
  • Gérer les ordres de travail (OT)
  • Visualiser l'état des pièces reçues
  • Localiser les kits sur un plan de stockage

Caractéristiques

  • Application web progressive (PWA)
  • Fonctionne sur mobile et desktop
  • Synchronisation en temps réel (Firebase)
  • Scanner d'étiquettes intégré (OCR)

Page Liste - Vue principale

2 / 8

Fonctionnalités

  • Affichage de tous les kittings
  • Filtrage par statut (Tous / Prêts / Incomplets)
  • Recherche par numéro d'ordre ou pièce
  • Statistiques en temps réel
  • Cartes colorées selon le statut

Code couleur

Vert = Kit complet, prêt à utiliser
Orange = Kit incomplet, pièces manquantes

Détail d'un Kitting

3 / 8

Informations affichées

  • Numéro d'ordre de travail (OT)
  • Description du kitting
  • Zone et emplacement de stockage
  • Date de réception des pièces
  • Liste des pièces avec quantités

Actions disponibles

  • Cocher les pièces reçues
  • Modifier les quantités
  • Changer la zone de stockage
  • Supprimer un kitting
  • Ajouter des pièces

Page Plan - Visualisation spatiale

4 / 8

Vue Plan (Desktop)

  • Plan 2D interactif de la zone de stockage
  • Zones déplaçables et redimensionnables
  • Sous-zones (étagères, cases)
  • Zoom avant/arrière
  • Image de fond personnalisable

Vue Mobile (arborescente)

  • Navigation par zones
  • Liste des sous-zones
  • Accès rapide aux kittings

Scanner d'étiquettes

5 / 8

Comment ça marche ?

  • Utilisez la caméra du téléphone
  • Photographiez l'étiquette de la pièce
  • L'OCR extrait automatiquement le numéro
  • Recherche automatique du kitting

Astuce

Pour de meilleurs résultats, assurez-vous que l'étiquette est bien éclairée et que le texte est lisible avant de prendre la photo.
